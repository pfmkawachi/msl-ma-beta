# MSL Motion Analyzer Beta

[![GitHub Pages](https://img.shields.io/badge/deployed-GitHub%20Pages-blue?style=for-the-badge&logo=github)](https://pfmkawachi.github.io/msl-ma/)
[![License: MIT](https://img.shields.io/badge/License-MIT-emerald?style=for-the-badge)](LICENSE)

スマートフォン、タブレット、PCのあらゆるデバイスでの動作分析を可能にする、高精度なWebベースのモーションアナライザーです。

サーバーへの動画アップロードは一切不要。完全ローカル処理（ブラウザ内完結）のため、大容量の動画でも一瞬で読み込め、プライバシーや通信量を気にする必要はありません。

## [beta v1.1 - 20260707] サーバーレス連携＆確実な録画機能の追加

### ✨ 新機能 (New Features)

*   **スマホ・PC間の完全ローカル接続 (QRコード連携)**
    *   外部のシグナリングサーバー（WebSocketやNode.js等）を一切使用せず、PC画面上のQRコードをスマホで読み取るだけでWebRTCの映像転送が開始されるようになりました（シグナリングレスWebRTC）。
    *   これにより、GitHub Pagesなどの静的ホスティング環境のみで、セキュアかつシームレスなローカル接続が完結します。
*   **Canvas連写方式による確実な録画・プレビュー (MediaRecorder非依存)**
    *   iOS/Safari等で挙動が不安定になりやすい `MediaRecorder` APIの使用を廃止しました。
    *   指定したFPS（24/30/60）に合わせて `<canvas>` に映像を直接描画し、メモリ上に静止画として連続保存する「高速連写方式」を採用しています。
    *   この設計により、動画ファイルのキーフレームに依存しないため、コマ落ちやズレが100%発生せず、正確な1フレームごとのコマ送り・逆再生・A-Bループ解析が保証されます。
*   **テイク（試技）の履歴・サムネイル管理**
    *   PC版で録画を停止するたびに、画面右下のパネルに一時的な履歴としてサムネイルがストックされます。
    *   複数回撮影した中から過去のテイクをワンタップで呼び出し、即座にプレビューしたり、解析モードへ取り込んだりすることが可能です。

### 📱 UI/UXの改善 (UI/UX Updates)

*   **PC版 (`pc.html`)**
    *   初期ランディング画面に「スマホと接続」ボタンを新設しました。
    *   接続中は、右パネルが「ストリーミング操作パネル」に切り替わり、録画・プレビュー・取り込み（IMPORT）・保存を直感的に操作できます。
*   **スマートフォン版 (`sp.html`)**
    *   初期ランディング画面に「PCと接続」ボタンを新設しました。
    *   QRコード読み取り時、あらかじめ配信用カメラ（背面・広角・高FPS）を起動した状態でスキャン枠をオーバーレイ表示する仕組みを採用。QR読み取り成功時にカメラの再起動による暗転やカクつきが発生せず、一瞬でシームレスに配信モードへ移行します。

### 🔧 技術的な変更・追加ライブラリ (Technical Changes)

今回の実装に伴い、以下のライブラリ（CDN）を新規導入しています。
*   **`pako`**: 膨大なテキストデータであるWebRTCの接続情報（SDP）を、QRコード化可能なサイズに圧縮・復元するために使用。
*   **`qrcode`**: PC側でのQRコード生成（Canvas描画）に使用。
*   **`jsQR`**: スマートフォン側でのQRコード読み取り（カメラ映像の解析）に使用。

## ✨ 主な機能

- **スマートデバイス最適化**: アクセスした端末（iOS / Android / PC）を自動で判別し、最適なUI（モバイル版 / デスクトップ版）へ自動振り分け。
- **高精度コマ送り**: 24 / 30 / 60 FPSに対応した「+1F / -1F」「+5F / -5F」のフレームシーク。
- **自由なズーム & パン（位置移動）**: 
  - スマホ・タブレット：直感的な2本指ピンチ・パン操作
  - PC：マウスホイールでのズーム ＆ ドラッグでの位置移動
- **A-B間ループ再生**: 指定した2点間（A点・B点）を自動で繰り返しループ再生。フォームの反復チェックに最適。
- **左右反転（ミラー機能）**: ワンタップで動画を左右反転。利き手などに合わせた分析が可能。
- **キーボードショートカット (PC/タブレット)**: `Space`で再生/停止、`矢印キー`で正確なコマ送り操作が可能。
- **スタンドアロン起動対応**: ホーム画面に追加することで、ブラウザのツールバーを非表示にし、ネイティブアプリのように全画面で起動可能。
